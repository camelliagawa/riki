# 力覚センサ CSV グラフビューア (ZEF / DynPick)

DPViewer などで取得した **6軸力覚センサ（ZEF シリーズ / DynPick）の CSV** を
ドラッグ＆ドロップするだけで、力（N）・モーメント（Nm）の時系列グラフを表示する
**単一HTMLアプリ**です。インストール不要・オフライン動作。

完成イメージ（上段：力 / 下段：モーメント / 横軸：時間）に合わせた2段グラフを描画し、
軸範囲・色・ラベルなどを画面上で自由に調整、PNG / SVG 画像として書き出せます。

---

## 使い方

1. `index.html` をダブルクリックしてブラウザで開く
   （Chrome / Edge 推奨。`vendor/plotly.min.js` を同梱しているのでネット接続は不要）
2. 画面に **CSV をドラッグ＆ドロップ**（またはクリックして選択）
3. グラフが自動表示されます。左のパネルで調整 → 画像形式（PNG / SVG）を選び、
   「力グラフを出力」「モーメントグラフを出力」「力＋モーメントを1枚で出力」で保存

> サンプル: `sample/sample_ZEF-6A100-4-RAD.csv`（180秒・36万行）で動作確認できます。

---

## 対応する CSV 形式

DPViewer / 同梱サンプルプログラムが出力する **ヘッダ無し** CSV を想定しています。

| 列 | 内容 | 例 |
|---|---|---|
| 1 | 経過時間（既定 ms、0.5ms刻み = 応答1000Hz） | `0000001.500` |
| 2 | レコード番号（0〜9の繰り返し） | `00009` |
| 3 | Fx 検出値 [LSB] | `8598` |
| 4 | Fy 検出値 [LSB] | `7519` |
| 5 | Fz 検出値 [LSB] | `8805` |
| 6 | Mx 検出値 [LSB] | `8593` |
| 7 | My 検出値 [LSB] | `8156` |
| 8 | Mz 検出値 [LSB] | `8034` |

- 末尾カンマ・空行・ヘッダ行は自動でスキップします。
- 時間列の単位（ms / s）は自動推定し、左パネルで切り替えできます。

## LSB → 荷重への換算

取扱説明書 3.3 の換算式に従います。

```
力 [N]      = ( 検出値 [LSB] − 零点出力 [LSB] ) ÷ 主軸感度 [LSB/N]
モーメント [Nm] = ( 検出値 [LSB] − 零点出力 [LSB] ) ÷ 主軸感度 [LSB/Nm]
```

- **主軸感度** の初期値は出荷特性データ（ZEF-6A100-4-RAD）の値を設定済みです。
  別のセンサを使う場合は「① 単位・換算」パネルの感度欄を、お手元の
  出荷特性データの値（または `p` コマンドの出力）に書き換えてください。
- **零点出力** の初期値は 8192 LSB（取説の零点出力 8192±655 の代表値）。
  無負荷区間がある測定では「**区間平均で自動ゼロ**」を使うと各軸の零点を
  実測の平均から自動設定でき、より正確です。
- 「LSB（生値）」表示にも切り替え可能です。

> 初期値のセンサ感度（ZEF-6A100-4-RAD）:
> Fx 64.650 / Fy 65.010 / Fz 65.860 [LSB/N] ／
> Mx 1630.250 / My 1630.250 / Mz 1572.000 [LSB/Nm]

## 主な機能

- **① 単位・換算** … N/Nm ⇔ LSB 切替、零点（共通／軸別／自動ゼロ）、主軸感度
- **② 軸** … 時間軸・力軸・モーメント軸それぞれ 自動／手動レンジ、時間単位、縦軸の基準（ゼロ中心／ゼロ起点）
- **向き補正（工程ごとに反転）** … 力の向きが逆になる工程を、時間境界で区切って符号反転し＋に揃える（境界の自動検出つき／反転軸は選択式）
- **工程ごとの自動ゼロ** … 工程間で零点がドリフトする場合に、各工程の無負荷区間からそれぞれ零点を取り直し、両工程の基準を0に揃える
- **非表示区間** … ツール反転中などの不要データを隠す。反転境界の前後 [s]、または「開始-終了」の区間（複数可）をグラフから非表示
- **③ 系列** … 軸ごとの色（カラーピッカー / 16進）・表示ON/OFF・線の太さ
- **④ ラベル・デザイン** … タイトル、各軸ラベル、フォントサイズ、グリッド／凡例
- **⑤ 描画・性能** … 大容量CSV向けの間引き点数、WebGL/SVG レンダラ切替
- **画像出力** … 力／モーメントを別々に、または1枚で。PNG（2倍解像度）／SVG（ベクタ）
- **設定保存** … 調整した設定をブラウザに保存／既定値へリセット
- ズーム・パン・凡例クリックで表示切替（Plotly のインタラクティブ操作）

## 補足・既知の制限

- 大容量CSV（数十万行）は「⑤ 描画・性能」の最大点数で間引いて表示します
  （既定 1系列あたり40,000点）。出力画像も間引き後の波形になります。
- ローカルファイルを `file://` で開く方式のため、ブラウザのセキュリティ制限は受けません。
- 列構成が上表と異なるCSV（列の並び替え・別フォーマット）には未対応です。
  必要であれば列マッピング機能を追加できます。
